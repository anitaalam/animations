CSS Animations

Transition and Transform

I created an interface to preview travel popular travel destinations. This animation was created using pseudo-classes to create interaction and pseudo-elements for positioning. Flexbox was used for alignment to create the 2x2 grid.

SVG Animation Line Art

I created an animated SVG line art using stroke-dasharray and stroke-dashoffset. I first created the "hello" in Procreate and transferred it to Illustrator where I used the pen tool to write out the word.

Scalable Preloader

This was a two part project. Part one was to create a preloader. Part two was to make it scalable by font-size and viewport size. I used @media to make it scalable.

JavaScript & GSAP Animations

Manipulating CSS with JavaScript

This is an introductory piece to using JavaScript to control CSS animation. An addEventListener() was used to change the backgrond color on click.

JavaScript Animate() Method

On the right side is an animation that I created using the JavaScript Animate() method to create the same CSS animation as the left side. Both side has the same animation, it's just that one side is created using CSS and the other JavaScript.

GSAP TimeLine

This is a fun animation I created using the GSAP library. Originally, tween controls were added to the play, pause, reverse and reset buttons. However, in part two, the tween animation was converted to a timeline animation using GSAP's timeline method.
